# Commando

Currently alpha quality

## Features
####`esc` key
- Commando roll outta UITextFields/UITextViews
- Commando roll outta UIAlertviews

####`f` key
- Find all tapable fields on screen - badass [vimium](http://vimium.github.io/) style
